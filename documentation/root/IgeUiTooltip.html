IgeUiTooltip CLASS

Declared in IgeUiTooltip.js

Description
Provides a UI tooltip. Change properties (textBox, fonts, backgroundcolor) at free will.
Extends
IgeUiElement

Methods

init

init({} parent, {} mountEntity, {} width, {} height, {} content)
  • {}parent Where the mousemove is captured i.e. on which element the tooltip should appear
  • {}mountEntity Where the tooltip should be mounted. A scene is suggested.
  • {}width Width of the tooltip
  • {}height Height of the tooltip
  • {}content The content which is set with public method "setContent". Can be string, array(2) or an entity
