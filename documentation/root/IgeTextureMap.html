IgeTextureMap CLASS

Declared in IgeTextureMap.js

Description
Texture maps provide a way to display textures / cells across a tile map.
Extends
IgeTileMap2d

Methods

autoSection

Gets / sets the auto sectioning mode. If enabled the texture map will render to off-screen canvases in sections denoted by the number passed. For instance if you pass 10, the canvas sections will be 10x10 tiles in size.
autoSection({Number} val)
  • {Number}val The size in tiles of each section.
Returns {*}

drawSectionBounds

Gets / sets the draw sections flag. If true the texture map will output debug lines between each section of the map when using the auto section system.
drawSectionBounds({Number} val)
  • {Number}val The boolean flag value.
Returns {*}

cacheForceFrame

Forces a cache redraw on the next tick.
cacheForceFrame()

negate

Takes another map and removes any data from this map where data already exists in the other.
negate({IgeTileMap2d} entity)
  • {IgeTileMap2d}entity The other map to read map data from.
Returns {*}

addTexture

Adds a texture to the texture map's internal texture list so that it can be referenced via an index so that the texture map's data will be something like [[textureId, textureCell]] or a real world example: [[0, 1], [1, 1]].
addTexture({IgeTexture} texture)
  • {IgeTexture}texture
Returns {Integer} The index of the texture you just added.

allTexturesLoaded

Checks the status of all the textures that have been added to this texture map and returns true if they are all loaded.
allTexturesLoaded()
Returns {Boolean} True if all textures are loaded, false if not.

paintTile

Sets the specified tile's texture index and cell that will be used when rendering the texture map.
paintTile({Number} x, {Number} y, {Number} textureIndex, {Number} cell)
  • {Number}x The tile x co-ordinate.
  • {Number}y The tile y co-ordinate.
  • {Number}textureIndex The texture index.
  • {Number}cell The cell index.

clearTile

Clears any previous tile texture and cell data for the specified tile co-ordinates.
clearTile({Number} x, {Number} y)
  • {Number}x The tile x co-ordinate.
  • {Number}y The tile y co-ordinate.

loadMap

Reads the map data from a standard map object and fills the map with the data found.
loadMap({Object} map)
  • {Object}map The map data object.

saveMap

Returns a map JSON string that can be saved to a data file and loaded with the loadMap() method.
saveMap()
Returns {Object} The map data object.

clearMap

Clears the tile data from the map effectively wiping it clean. All existing map data will be removed. The textures assigned to the texture map will not be affected.
clearMap()
Returns {*}

reset

Clears tile data from the map and also removes any textures from the map that were previously assigned to it. This is useful for reverting the texture map to it's virgin state as if it had just been created.
reset()
Returns {*}

tileTextureIndex

Gets / sets the specified tile's texture index.
tileTextureIndex({Number} x, {Number} y, {Number} textureIndex)
  • {Number}x The tile x co-ordinate.
  • {Number}y The tile y co-ordinate.
  • {Number}textureIndex The new texture index.

tileTextureCell

Gets / sets the specified tile's texture cell.
tileTextureCell({Number} x, {Number} y, {Number} cell)
  • {Number}x The tile x co-ordinate.
  • {Number}y The tile y co-ordinate.
  • {Number}cell The new cell index.

convertHorizontalData

Converts data that is saved in the format [x][y] to the IGE standard of [y][x] and then returns the data.
convertHorizontalData({Array} mapData)
  • {Array}mapData The map data array.
Returns {Object} The new map data.

tick

Handles rendering the texture map during engine tick events.
tick({CanvasRenderingContext2d} ctx)
  • {CanvasRenderingContext2d}ctx

_ensureSectionExists

Private method, checks if the specified section currently exists in the cache and if not, creates it.
_ensureSectionExists({Number} sectionX, {Number} sectionY)
  • {Number}sectionX The section's x co-ordinate.
  • {Number}sectionY The section's y co-ordinate.

_drawSectionsToCtx

Private method, draws cached image sections to the canvas context.
_drawSectionsToCtx({CanvasRenderingContext2d} ctx)
  • {CanvasRenderingContext2d}ctx

_renderTile

Private method, renders a tile texture based on data from the texture map, to a cached section.
_renderTile({CanvasRenderingContext2d} ctx, {Number} x, {Number} y, {Object} tileData, {Object} tileEntity, {IgeRect} rect, {Number} sectionX, {Number} sectionY)
  • {CanvasRenderingContext2d}ctx
  • {Number}x The tile x co-ordinate.
  • {Number}y The tile y co-ordinate.
  • {Object}tileData The tile's texture and cell data.
  • {Object}tileEntity The object that represents the tile.
  • {IgeRect}rect The rectangular area to limit drawing to.
  • {Number}sectionX The x co-ordinate of the section to draw to.
  • {Number}sectionY The y co-ordinate of the section to draw to.
Returns {*}

_newTileEntity

Private method, creates an entity object that a texture can use to render itself. This is basically a dummy object that has the minimum amount of data in it that a texture requires to render such as geometry, texture cell and rendering position.
_newTileEntity()
Returns {Object} The new tile entity object.
