IgeTweenComponent CLASS

Declared in IgeTweenComponent.js

Description
This component is already included in the IgeEngine (ige) instance and is not designed for use in any other way! It handles global tween processing on all tweening values.
Extends
IgeClass

Methods

start

Start tweening particular properties for the object.

Returns {Number} The index of the added tween or -1 on error.

  • {IgeTween}tween The tween to stop.
