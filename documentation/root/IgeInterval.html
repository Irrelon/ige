IgeInterval CLASS

Declared in IgeInterval.js

Description
Provides an alternative to setInterval() which works based on the engine's internal time system allowing intervals to fire correctly, taking into account pausing the game and differences in rendering speed etc.
Extends
IgeEventingClass

Methods

init

Creates a new timer that will call the method every given number of milliseconds specified by the interval parameter.
init({Function} method, {Number} interval)
  • {Function}method The method to call each interval.
  • {Number}interval The number of milliseconds between each interval.

addTime

Adds time to the timer's internal clock.
addTime({Number} time)
  • {Number}time The time in milliseconds to add to the timer's internal clock.
Returns {*}

cancel

Cancels the timer stopping all future method calls.
cancel()
Returns {*}

Checks for a timer event to see if we should call the timer method. This is called automatically by the IgeTimeComponent class and does not need to be called manually.
