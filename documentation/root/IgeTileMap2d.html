IgeTileMap2d CLASS

Declared in IgeTileMap2d.js

Description
Tile maps provide a way to align mounted child objects to a tile-based grid. NOTE: These are not to be confused with IgeTextureMap's which allow you to paint a bunch of tiles to a grid.

_objectOccupyTile

This method is attached to objects that are mounted to the tile map. Adds the object to the tile map at the passed tile co-ordinates. If no tile co-ordinates are passed, will use the current tile position and the tileWidth() and tileHeight() values.
_objectOccupyTile({Number} x, {Number} y, {Number} width, {Number} height)
  • {Number}x X co-ordinate of the tile to occupy.
  • {Number}y Y co-ordinate of the tile to occupy.
  • {Number}width Number of tiles along the x-axis to occupy.
  • {Number}height Number of tiles along the y-axis to occupy.

_objectUnOccupyTile

This method is attached to objects that are mounted to the tile map. Removes the object from the tile map at the passed tile co-ordinates. If no tile co-ordinates are passed, will use the current tile position and the tileWidth() and tileHeight() values.
_objectUnOccupyTile({Number} x, {Number} y, {Number} width, {Number} height)
  • {Number}x X co-ordinate of the tile to un-occupy.
  • {Number}y Y co-ordinate of the tile to un-occupy.
  • {Number}width Number of tiles along the x-axis to un-occupy.
  • {Number}height Number of tiles along the y-axis to un-occupy.

_objectOverTiles

This method is attached to objects that are mounted to the tile map. Returns an array of tile co-ordinates that the object is currently over, calculated using the current world co-ordinates of the object as well as it's 3d geometry.
_objectOverTiles()
Returns {Array} The array of tile co-ordinates as IgePoint instances.
