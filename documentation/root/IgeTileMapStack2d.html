IgeTileMapStack2d CLASS

Declared in IgeTileMapStack2d.js

Description
Tile maps provide a way to align mounted child objects to a tile-based grid. NOTE: These are not to be confused with IgeTextureMap's which allow you to paint a bunch of tiles to a grid.

IgeEntity

_objectOccupyTile

Attached to objects that are mounted to the tile sheet. Adds the object to the tile map at the passed tile co-ordinates. If no tile co-ordinates are passed, will use the array returned from calling overTiles() and will add the object to each tile the object is "over".
_objectOccupyTile({Number} x, {Number} y, {Number} width, {Number} height)
  • {Number}x
  • {Number}y
  • {Number}width
  • {Number}height

_objectUnOccupyTile

Attached to objects that are mounted to the tile sheet. Removes the object from the tile map at the passed tile co-ordinates. If no tile co-ordinates are passed, will use the array returned from calling overTiles() and will remove the object from each tile the object is "over".
_objectUnOccupyTile({Number} x, {Number} y, {Number} width, {Number} height)
  • {Number}x
  • {Number}y
  • {Number}width
  • {Number}height

_objectOverTiles

Attached to objects that are mounted to the tile sheet. This method returns an array of tile co-ordinates that the object is currently over, calculated using the current world co-ordinates of the object as well as it's 3d geometry.
_objectOverTiles()
Returns {Array} The array of tile co-ordinates as IgePoint3d instances.
